# 🚀 个人博客后端接口部分 MidWay.js


## ❤️MidWay快速入门

参见 [https://midwayjs.org/docs/intro]

### 启动配置

```bash
$ npm i
$ npm run dev
$ open Swagger文档 http://localhost:8888/swagger-ui/index.html
```
